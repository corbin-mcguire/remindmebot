# Remind me, Bot!

Offload your thoughts into me and I'll remind you about them when you need me to!

## Usage

```
/remindme 20 minutes "Message"
```

### From source

```bash
> npm install
> npm run serve
```

### Docker

```bash
> docker build . -t remindmebot
> docker run -p 49160:8080 -d remindmebot
```
